# Security Research Projects

## Vulnerability Research

### DrivePulse SQL Injection Vulnerability

**Status:** Public Disclosure

**Severity:** High

**CWE:** CWE-89 (SQL Injection)

**Affected Component:**

```text
admin/sortData/fetch.php
```

**Research Summary**

During source-code review of the DrivePulse Driving School Management System, multiple user-controlled parameters were found to be incorporated into SQL queries without proper parameterization.

Potential impact includes:

- Data disclosure
- Administrative account enumeration
- Business data tampering
- Database compromise

**Research Documentation**

- Vulnerability Analysis:
  - https://github.com/Earlyde/cve/issues/1

**Focus Areas**

- Root Cause Analysis
- Secure Coding Review
- Defensive Remediation
- Database Security
